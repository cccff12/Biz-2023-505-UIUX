처음으로 만드는 HTML 문서로 작성된 Web Page, Web Page는 HTML형식의 문서로 작성한다 HTML 문서에서 p tag 로 시작하면 하나의 문단이 시작된다. 하나의 문단은 close /p 이며 문단이 끝났음을 나타낸다 문단 내에서는 아무리 Enter 를 입력해도 브라우저에 나타날떄는 모두 무시된다. 단 enter를 입력하면 한 칸의 빈칸이 나타난다. 문단 중간에서 여러개의 빈칸(Space)를 입력하여도 한개의 빈칸만 남기고 모두 무시된다.
